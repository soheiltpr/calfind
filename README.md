## کال‌فایند (Calfind)

وب‌اپلیکیشنی برای هماهنگی زمان جلسات گروهی با پشتیبانی کامل از زبان فارسی، تقویم شمسی و همگام‌سازی لحظه‌ای. ادمین برای هر پروژه یک لینک اختصاصی ایجاد می‌کند و اعضا می‌توانند بازه‌های زمانی آزاد خود را ثبت کنند. نتایج برای تمام افراد حاضر در صفحه به صورت زنده به‌روزرسانی می‌شود.

## پیشنیازها

- Node.js نسخه 18 یا بالاتر
- حساب Supabase با قابلیت Realtime (پلن رایگان کافی است)

## راه‌اندازی محلی

1. پکیج‌ها را نصب کنید:

   ```bash
   npm install
   ```

2. در Supabase پروژه بسازید و جدول‌ها و سیاست‌های دسترسی را اعمال کنید. فایل `supabase/schema.sql` را می‌توانید در تب SQL Editor اجرا کنید.

3. کلیدهای عمومی Supabase را در `env.example` کپی کرده و نام فایل را به `.env.local` تغییر دهید:

   ```
   NEXT_PUBLIC_SUPABASE_URL="https://<your-project>.supabase.co"
   NEXT_PUBLIC_SUPABASE_ANON_KEY="<anon-key>"
   ```

4. سرور توسعه را اجرا کنید:

   ```bash
   npm run dev
   ```

5. به `http://localhost:3000` مراجعه کنید. صفحه اصلی فرم ساخت پروژه را نمایش می‌دهد. پس از ایجاد پروژه، لینک `/project/<id>` را با اعضا به اشتراک بگذارید.

## معماری و ویژگی‌ها

- **Next.js 14 (App Router)** با TypeScript و Tailwind
- **Supabase Realtime** برای همگام‌سازی زنده پاسخ‌ها
- **react-multi-date-picker** با تقویم فارسی برای انتخاب تاریخ
- **رنگ‌بندی پویا** بر اساس تعداد افراد حاضر در هر بازه
- **فونت Vazirmatn** و راست‌چین کامل رابط کاربری

## بررسی سریع

- `src/app/page.tsx`: صفحه اصلی و فرم ساخت پروژه
- `src/app/project/[id]/page.tsx`: صفحه تعامل اعضا با پروژه
- `src/components`: فرم‌ها و خلاصه‌سازی داده‌ها
- `supabase/schema.sql`: اسکریپت ساخت جدول‌ها و سیاست‌های RLS

## استقرار

می‌توانید پروژه را در هر پلتفرم سازگار با Next.js (مانند Vercel) مستقر کنید. پس از استقرار، متغیرهای محیطی Supabase را در تنظیمات پلتفرم میزبان نیز ست کنید.
